简数(keydatas.com)数据采集平台是一个完全在线配置和云端采集的网页数据采集和发布平台，它功能强大，操作简单，不仅提供网页内容采集、数据处理和发布等数据采集基本功能，还创新实现了智能提取引擎以及书签一键采集发布等特色功能等。支持微信公众号文章、淘宝客商品采集。


== Description ==

简数(<a href="http://www.keydatas.com" target="_blank">keydatas.com</a>)网页数据采集发布插件主要功能特性：<br>
1.采集不需安装任何客户端或插件，不用研究网页源代码，在线选取、点击，保存，就可以实现采集；<br>
2.支持微信公众号文件采集，仅需输入微信公众号名称或ID或者关键词即可采集；<br>
3.智能识别数据和规则，包括：列表页、翻页和详情页（标题、正文、作者、发布时间等）；<br>
4.书签一键采集发布（独家）:随时随地，看到好的文章，点击浏览器书签即可把第三方网页内容采集并发布到您的网站！<br> 
5.更多特性：<br>
支持淘宝联盟商品采集；<br>
定时采集更新；<br>
灵活的数据二次加工处理；文章SEO优化（同义词替换、简体繁体转换、正文中间插入其它内容等）；<br>
支持文件图片资源下载，突破防盗链；<br>
轻松发布到WordPrss和织梦等；<br>
多节点分布式采集，免费IP代理等机制。<br>
相比火车头、八爪鱼、wpspider、WP-AutoPost、小蜜蜂，简数数据采集平台更加简单易用，功能同样强大，且没有繁杂的流程。

安装：
在WordPress插件后台搜索 keydatas 或 “简数” 或 “数据采集” 安装启用即可
更多详情请访问<a href="http://www.keydatas.com" target="_blank">简数官网</a>

== Frequently Asked Questions ==
<h2>旧版本1.x.x（手工上传安装方式）的卸载</h2>
请先停用1.x.x的旧版本，然后删除，如果出现“未能完整移除插件keydatas/keydatas.php”，请到/wp-content/plugins目录下删除keydatas，然后刷新wordpress插件页面即可。
